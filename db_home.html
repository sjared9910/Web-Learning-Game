Group 4 Entertainment

This application allows for the populating of the database of questions and answers that will be pooled for the knowledge game. Webgame will test on different subjects with questions within each. First, create the subjects to test on using the "New Subject" tool and follow that using the "New Question" tool for placing questions within each subject. Questions and Subjects can be deleted at any time. Use the view Questions tool to view the contents of the database including subjects and questions within.
